📦 cesium-utils-lib 示例

每个示例是一个独立的 HTML 文件，可直接打开或通过 npx vite examples --open 运行

01 鼠标点击获取经纬度
getLonLat — 点击地图任意位置，显示经纬度与高度
 02 三维坐标转屏幕坐标
cartesianToScreen — 3D 位置跟随 HTML 标签，拖拽地图实时更新
 03 距离测量与中点
distance + midpoint — 点击两个位置，显示球面距离和中点坐标
 04 扩散扫描圆环
createScanCircle — 可调颜色/半径/速度/多环间隔的雷达扫描动画
 05 截图工具
screenshot — 高清截图 + 双击渲染 + 自动下载，支持 1x/2x/3x
 06 自定义标记
createPinImage + createBillboardEntity — 点击添加编号标记，支持颜色/大小
 07 多点视野飞行
flyToPositions — 预设航线/手动标点，一键飞至所有点可见视野
 08 相机控制
getCameraHeight + setView + flyTo — 实时高度 + 一键跳转 + 变焦
